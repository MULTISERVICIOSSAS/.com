Conocer el proceso

1. Solicitud

El cliente registra la solicitud en la plataforma y recibe orientacion por WhatsApp.

2. Curso

Acceso a módulos digitales de manipulación higiénica de alimentos.

3. Examen

Evaluacion final con resultado guardado en la base central para revision administrativa.

4. Certificado

Generación de PDF y validación por código en línea.

Confianza

Preguntas frecuentes

Respuestas claras para que el usuario entienda el servicio, la validación y los límites de la plataforma.

¿El curso es digital?

Sí. La experiencia está preparada con módulos digitales, evaluación final y acompañamiento comercial por WhatsApp.

¿Cómo recibo el certificado?

Después de completar el proceso, el área administrativa genera el certificado PDF con código único y datos del titular.

¿Cómo se valida un certificado?

La página de validación permite consultar por código único o documento parcial, sin exponer documentos completos.

¿Cómo se publica un certificado nuevo?

El administrador genera el certificado y exporta la base pública para que el código quede disponible en la validación.
